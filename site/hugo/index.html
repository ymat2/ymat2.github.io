Hugo - 静的サイトを高速生成

https://gohugo.io/about/

インストール

homebrewで一発。

brew install hugo

Quick Start

  1. サイトの骨組みを作る。

    mkdir PATH_TO_SITE
cd PATH_TO_SITE
hugo new site .

  2. 生成される./content/ディレクトリの中に記事を書いていく。

    hugo new content/about.md

    ---
title: "デモページ"
date: 2022-11-03T22:09:12+09:00
---

これは**テストページ**です。

  3. サーバーを走らせてhttp://localhost:1313/aboutにアクセス。

    hugo server -D -w .
